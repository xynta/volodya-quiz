# Викторина «Кто хочет стать миллионером» — «Званый ужин»

Квиз-лесенка из 15 вопросов по мотивам недели «Званого ужина» (РЕН ТВ).
Каждая игра — случайный вечер с одним из пяти героев недели (Ольга Стукалова,
Александр Асиновский, Лера Гаврилова, Владимир Чони, Владимир Алексеев).
Денежная лесенка от 100 ₽ до 1 000 000 ₽, несгораемые уровни 5 (1 000 ₽) и
10 (32 000 ₽), три подсказки — 50:50, помощь зала, звонок другу. Все вопросы —
в [data/questions.json](data/questions.json).

Две версии с общим игровым ядром (`game/`):

- **Telegram-бот** — [bot.py](bot.py) (aiogram). Нужен `BOT_TOKEN` в `.env`.
- **Консольная версия в ASCII-стиле** — [console_quiz.py](console_quiz.py).
  Токен не нужен.

## Консоль — быстрый старт

```bash
python3 console_quiz.py
```

Управление: `A/B/C/D` — ответ, `50` / `зал` / `друг` — подсказки,
`л` — показать лесенку, `в` — выход.

## Сборка в исполняемый файл (без Python у игрока)

- **Windows (.exe):** двойной клик по [build.bat](build.bat) →
  `dist\volodya_quiz.exe`
- **macOS / Linux:** `./build.sh` → `dist/volodya_quiz`
- **Облачная сборка** обеих версий — GitHub Actions, вкладка *Actions*.

Подробности: [BUILD.md](BUILD.md).

## Telegram-бот

```bash
pip install -r requirements.txt
cp .env.example .env   # вписать BOT_TOKEN от @BotFather
python3 bot.py
```

### Скачивание билдов через бота (для админа)

Скрытая команда `/builds` присылает готовые консольные сборки прямо в чат.
Доступна только пользователям из `ADMIN_USERNAMES` (по умолчанию — `@Ignodeau`,
меняется в `.env`). Бот ищет исполняемые файлы в папках `dist/` и `artifacts/`
рядом с собой — туда кладут локальную сборку (`./build.sh`) или скачанные
артефакты GitHub Actions. Файлы до 50 МБ Telegram отдаёт без проблем.
